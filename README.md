# detect_activity

TinyML: detect activity by accelerometer and gyroscope.

## Dataset

This project uses the **Human Activity Recognition Using Smartphones Dataset** from the UCI Machine Learning Repository.

**Source:** https://archive.ics.uci.edu/ml/datasets/human+activity+recognition+using+smartphones

**Authors:** 
- Davide Anguita, Alessandro Ghio, Luca Oneto, Xavier Parra, Jorge L. Reyes-Ortiz
- Smartlab - Non-Linear Complex Systems Biology and Engineering Lab (NLCSE)
- Department of Informatics, BioEngineering, Robotics, Systems Engineering (DIBRIS)
- University of Genova, Italy

**Citation:**

This project uses the SIDDHA dataset from [neuromorphic-polito/siddha](https://huggingface.co/datasets/neuromorphic-polito/siddha) on Hugging Face, which is distributed under the MIT License.
